# MultiMiner

> **Summary:** Mine smarter, not harder. Vein mine ores and fell entire trees with a single break.

---

## ⛏️ What is MultiMiner?

**MultiMiner** is a lightweight Fabric mod that adds **vein mining** and **tree felling** to Minecraft — no more breaking blocks one by one.

Break one ore, and the entire vein comes with it. Chop one log, and the whole tree comes down. Simple, clean, and fully configurable.

---

## ✨ Features

### Vein Mining
- Break a connected group of ores all at once
- Works with all vanilla ores (and modded ores if compatible)
- Activate by sneaking, holding a keybind, or always-on — your choice

### Tree Felling
- Chop down entire trees by breaking a single log
- Leaves decay naturally after felling
- Supports all vanilla tree types

### In-Game Config Menu
- Fully customize behavior without touching config files
- Toggle vein mining and tree felling independently
- Set max block limits per action
- Choose activation mode (sneak, keybind, or always-on)
- Adjust which blocks are affected

## ⚙️ Configuration

MultiMiner comes with a built-in config screen accessible from Mod Menu. You can customize:

- **Max vein size** — How many blocks can be mined at once (default: 64)
- **Max tree size** — How many logs can be felled at once (default: 128)
- **Activation mode** — Sneak, keybind, or always active
- **Tool requirement** — Require the correct tool or allow bare hands
- **Exhaustion multiplier** — Control how much hunger vein mining/felling costs
- **Block whitelist/blacklist** — Fine-tune which blocks are affected
